# Neon Blocks

Neon Blocks is a responsive Next.js Tetris-style game with a sleek cyberpunk aesthetic.

## Features
- Endless gameplay with scoring for cleared lines
- Three difficulty levels that adjust falling speed
- Keyboard and touch controls
- Neon-inspired visual style with glowing accents

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm start
```
